# IMDB-Scraping-Project
This aim of this project was to scrape the information about Top 250 rated movies on IMDB and visualize their diversity.

Attributes scraped:
1. Movie Name
2. Director
3. Country
4. Language
5. Running Time
6. Release Date
8. IMDB Rating

Library Used:
1. BeautifulSoup
2. Urllib
3. Pandas
4. Numpy
5. Matplotlib.pyplot
6. seaborn
8. re
